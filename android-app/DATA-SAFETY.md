# Data Safety Draft

Use this as a practical draft for the Google Play Console Data Safety section.

## Data Collection

- Personal info: No
- Financial info: No
- Messages: No
- Photos and videos: No
- Audio files: No
- Files and docs: No
- Calendar: No
- Contacts: No
- App activity: No
- Web browsing: No
- App info and performance: No
- Device or other IDs: No
- Location: No

## Data Sharing

- Shared with third parties: No

## Security

- Data encrypted in transit: Yes
- Data deletion request: Not applicable

## Permission Use Summary

- Camera:
  used only for QR scanning
- Photos / media:
  used only when the user selects an image for QR reading
- Internet:
  used to load secure web app content

## Important Note

If the website later adds analytics, ads, login systems, or any external user-data collection, update the Play Console Data Safety answers accordingly.
